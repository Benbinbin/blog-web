# 图标字体

图标字体是一些矢量图标，通过引入 CSS 样式表的方式加载，可用于网页中的小图标，类似于字体可以设置大小和颜色。

最常用的图标字体库是 Font Awesome，它提供了十分多的矢量图标。

  1. 通过 CDN 方式在 HTML 文档中引入 Font Awesome 样式表，推荐使用国内托管的 CDN 源
    <link href="https://cdn.bootcss.com/font-awesome/5.13.0/css/all.css" rel="stylesheet">
    1
    💡 根据官网推荐可以使用 Font Awesome Kit
  2. 通过关键字或浏览官网查找需要的图标（可免费使用的图标是深色的，在账号升级时需要高级用户才可以的图标显浅灰色），复制相应的代码插入到网页中，如 <i class="fas fa-camera"></i> 在网页插入一个照相机图标（行内元素）
