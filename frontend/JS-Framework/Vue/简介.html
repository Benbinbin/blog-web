# 简介

参考：

  • Vue 教程
  • Vue 出一個電商網站

Vue 是一个用于构建用户界面的渐进式 JavaScript 框架，类似于 MVVM 模型，通过简洁的模板语法允许开发者声明式地将 DOM 与底层 Vue 实例的数据进行绑定，其双向绑定的特点可以实现以数据驱动视图的生成。

在底层的实现上，Vue 将模板编译成虚拟 DOM 渲染函数，结合响应系统 Vue 能够智能地计算出最少需要重新渲染多少组件，并把 DOM 操作次数减到最少。

官方推出整合套件 Vue Cli （基于 Webpack）可以更方便地实现项目的打包。

# 配置环境

如果使用 Visual Studio Code 作为开发编辑器，推荐安装插件以提高开发效率：

  • Live Server 创建本地服务器以浏览项目
  • vue 高亮 Vue 代码
  • Vue 2 Snippets Vue 代码片段补全

💡 由于 Vue 是由数据驱动生成页面的（以双向绑定的方式实现 model 和 view 两者的实时更新），调试过程中数据会不断变动，使用 Vue 官网提供的开发者工具Vue Devtools（ Chrome 插件 ）实时解析页面的数据，可以更方便地进行测试。

# 安装 Vue

Vue 提供开发版本和生产版本两种版本，在开发环境中应该选择择开发版本（完整版），以获取完整的警告反馈信息和调试模式。

💡 生产版本 vue.min.js 删除了警告的压缩版本，文件体积更小，约 33.30KB，适合项目正式部署发布时使用。

Vue 安装可以通过直接下载 Vue 脚本到本地使用，或通过 NPM 安装到项目目录中（国内开发建议将 NPM 源设置为国内的镜像以提升安装速度），以便更好地与 webpack 或 Browserify 模块打包器工具配合使用。
