Redis

2021/7/6 - Update 2021/7/6

Redis 是 Web server 最常用的缓存数据库，它将数据存放在内存中，相比 MySQL 访问速度快（内存和硬盘的区别），但成本较高，可存储的数据量较小。

#redis

session

2021/7/6 - Update 2021/7/6

session 是暂存在服务器中的数据，它在浏览器（客户端）的 cookie 中只保存相应的 session id，这样可以将临时但敏感的数据存储在服务器端，避免了客户端的数据泄露或被操作。

核心思想

2021/7/6 - Update 2021/7/6

TailwindCSS 核心思想：Tailwind 提供了多种基础类，这些基础类按照规则可以用横线 - 相连进行组合，构成特点的语义，将相应的 UI 样式应用在 HTML 元素上

#CSS#TailwindCSS

简介

2021/7/6 - Update 2021/7/6

Tailwind 是一个 CSS UI 框架，它最大的特点是遵循基础/通用类优先 utility-first 的原则，它只提供了基础的工具类 utility classes（如内边距 padding、字体 text 和 font、动效 transition 等预设类），可以让你直接在 HTML 源码上使用这些基础类构建一个完全定制化的设计。
