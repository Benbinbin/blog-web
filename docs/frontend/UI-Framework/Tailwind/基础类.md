# 基础类
Tailwind 提供了大量开箱即用的基础类，将它们添加到 HTML 元素类名中，可以在最终的样式表中编译出相应的样式

## 布局

### Aspect Ratio
[Aspect Ratio 基础类](https://tailwindcss.com/docs/aspect-ratio)设置元素的宽高比例，一般用于设置图片 `<img>` 或 视频 `<video>` 元素

:bulb: 支持使用任意值 `aspect-[4/3]`

:bulb: 可以在 Tailwind 的配置文件的 `theme.aspectRatio` 或 ``theme.extend.aspectRatio` 进行定制

### Container
[Container 基础类](https://tailwindcss.com/docs/container)根据页面所处的断点设置元素/容器的宽度最大值

该基础类默认并不会将元素/容器居中，可以在 `theme.container` 属性中进行开启

```js
// 📄 tailwind.config.js
module.exports = {
  theme: {
    container: {
      center: true,
    },
  },
}
```

如果希望在元素/容器两侧添加水平内边距，可以在 `theme.container` 属性中进行设置

```js
// 📄 tailwind.config.js
module.exports = {
  theme: {
    container: {
      padding: '2rem',
    },
  },
}
```

也可以为每个断点下的容器添加不同的内边距

```js
// 📄 tailwind.config.js
module.exports = {
  theme: {
    container: {
      padding: {
        DEFAULT: '1rem',
        sm: '2rem',
        lg: '4rem',
        xl: '5rem',
        '2xl': '6rem',
      },
    },
  },
};
```

### Columns
[Columns 基础类](https://tailwindcss.com/docs/columns)为元素/容器设置列布局，可以实现容器内子元素类似 grid 布局的多列的排布，但它更简便，可以方便地实现类似杂志多列段落的排版布局。

由于该基础类对应的样式属性是一个[简写属性](https://developer.mozilla.org/zh-CN/docs/Web/CSS/columns)，所以该基础类分为两种类型，前缀 `columns-` 接着的是整数，表示列数，如 `columns-3`，即容器应该分为 3 列；前缀 `columns-` 接着的是表示每一列的宽度的值，如 `columns-xs`，即表示每一列的宽度是 `20rem`，列数会基于容器的总宽度自动计算出来。

:bulb: 可以设置列间隙，使用 [Gap](https://tailwindcss.com/docs/gap) 基础类，它会为各列之间添加水平间隙

:bulb: 支持使用任意值 `columns-[10rem]`

:bulb: 可以在 Tailwind 的配置文件的 `theme.columns` 或 `theme.extend.columns` 进行定制

### Break
[Break After](https://tailwindcss.com/docs/break-after)、[Break Before](https://tailwindcss.com/docs/break-before)、[Before Inside](https://tailwindcss.com/docs/break-inside) 这三个基础类，控制元素在打印时的前后或内部是否要换页

### Box Decoration Break
[Box Decoration Break](https://tailwindcss.com/docs/box-decoration-break) 基础类用以定义当元素跨多行、多列、多页时，元素的片段（每一行、每一列、每一页）应如何呈现

* 基础类 `.box-decoration-clone` 生成的样式是 `box-decoration-break: clone;` 每一行看作一个独立元素，它们的样式都一样 clone 原样式
* 基础类 `.box-decoration-slice` 生成的样式是 `box-decoration-break: slice;` 连续多行作为一个连续整体，每一行作为其中一个片段

:clapper: 具体效果可以查看这个[视频教程](https://youtu.be/C9ccfaIlqD4)

### Box Sizing
[Box Sizing](https://tailwindcss.com/docs/box-sizing) 基础类设置元素的盒子模型

:bulb:Tailwind 在重置样式 preflight 中将所有元素设置为奇异盒子模型，其样式为 `box-sizing: border-box;`

### Display
[Display](https://tailwindcss.com/docs/display) 基础类用于设置元素的 `display` 样式属性

* `flow-root` 基础类用于为元素（模拟根元素）创建 BFC 以包含内部所有元素，可用于清除浮动

* [`contents` 基础类](https://tailwindcss.com/docs/display#contents)将元素设置为「影子」容器，其子元素就像是其父元素的直接子元素一样

* 对于表格布局，Tailwind 提供了多个相关的基础类 `table`、`.table-row`、`.table-cell`、`.table-caption`、`.table-column`、`.table-column-group`、`.table-header-group`、`.table-row-group` 和 `.table-footer-group` 可以创建一个响应式的表格

```html
<div class="table w-full ...">
  <div class="table-header-group ...">
    <div class="table-row">
      <div class="table-cell text-left ...">Title</div>
      <div class="table-cell text-left ...">Artist</div>
      <div class="table-cell text-left ...">Year</div>
    </div>
  </div>
  <div class="table-row-group">
    <div class="table-row">
      <div class="table-cell ...">Chocolate Starfish And The Hot Dog Flavored Water</div>
      <div class="table-cell ...">Limp Bizkit</div>
      <div class="table-cell ...">2000</div>
    </div>
    <div class="table-row">
      <div class="table-cell ...">Significant Other</div>
      <div class="table-cell ...">Limp Bizkit</div>
      <div class="table-cell ...">1999</div>
    </div>
  </div>
</div>
```

* `hidden` 基础类所对应的样式是 `display: none` 它会将元素从页面移除，如果只希望隐藏元素，可以使用 [`.invisible` 基础类](https://tailwindcss.com/docs/visibility#invisible)

### Floats and Clear
[Floats 基础类](https://tailwindcss.com/docs/float)用于设置元素的浮动属性

[Clear 基础类](https://tailwindcss.com/docs/clear)用于清除该元素上方浮动元素，这样就可以避免产生类似杂志的文字环绕图片的布局

### Isolation
[Isolation 基础类](https://tailwindcss.com/docs/isolation)用于控制元素是否必须创建一个新的层叠上下文，这会影响[背景混合](https://developer.mozilla.org/zh-CN/docs/Web/CSS/background-blend-mode)的效果。

### Object
[Object Fit 基础类](https://tailwindcss.com/docs/object-fit)用于控制可替换元素，如 `<img>` 元素，其内容应该如何适应该元素的框（元素的盒子模型限制了宽高时）。

* `.object-contain` 基础类将内容进行保持其宽高比的缩放，以完整地显示在元素的盒子内
* `.object-cover` 基础类将内容进行保持其宽高比的缩放，以布满元素的盒子，因此可能对内容进行裁剪
* `.object-fill` 基础类可以将内容进行拉伸的缩放，以布满元素的盒子
* `.object-none` 基础类将内容保持其原有的尺寸来填充元素的盒子
* `.object-scale-down` 基础类将基于内容在 `object-fit: none` 或 `object-fit: contain` 两种情况下，以尺寸较小的情况来填充元素的盒子

[Object Position 基础类](https://tailwindcss.com/docs/object-position)配合以上的基础类使用，用以决定内容缩放后采用哪个部分来填充元素的盒子，一般用 `.object.center` 基础类，将内容的中间与元素的盒子中间重叠放置

:bulb: Tailwind 默认提供了 9 种位置放置方式，可以在 Tailwind 的配置文件的 `theme.extend.objectPosition` 属性中进行定制

:bulb: 支持使用任意值 `object-[center_bottom]`

### Overflow
[Overflow 基础类](https://tailwindcss.com/docs/overflow)用于控制当容器内的元素大于容器的尺寸时，应该如何处理。


### Overscroll
[Overscroll Behavior 基础类](https://tailwindcss.com/docs/overscroll-behavior)用以控制当可滚动区域到达底部时，用户如果继续滚动/过度滚动，页面（也可滚动时）的响应效果

> 移动端浏览器倾向于提供一种「触底」效果，甚至进行页面刷新。你可能也发现了，当对话框中含有可滚动内容时，一旦滚动至对话框的边界，对话框下方的页面内容也开始滚动了——这被称为「滚动链」。
*参考：[overscroll-behavior](https://developer.mozilla.org/zh-CN/docs/Web/CSS/overscroll-behavior)*

* 在某些情况下我们不想在过度滚动而触发页面随之滚动，可以使用 `.overscroll-none` 基础类来去除不需要的滚动链

* 但是在避免过度滚动引起页面随之滚动的同时，还想保留容器内的触底刷新或触底反弹的效果，可以使用 `.overscroll-contain` 基础类

### Position
[Position 基础类](https://tailwindcss.com/docs/position)用于设置元素的绝对定位样式属性

一般配置 [Top/Right/Bottom/Left 基础类](https://tailwindcss.com/docs/top-right-bottom-left)使用，以设置元素采用绝对定位时的位置

* `.inset-0` 基础类表示的样式

  ```css
  top: 0px;
  right: 0px;
  bottom: 0px;
  left: 0px;
  ```

* 支持负值 `.-left-4`

:bulb: Top/Right/Bottom/Left/Inset 基础类继承 Spacing 基础类以及 `auto` 和 `full`，可以间接在 Tailwind 配置文件的 `theme.spacing` 或 `theme.extend.spacing` 属性中定制

```js
// 📄 tailwind.config.js
module.exports = {
  theme: {
    extend: {
      spacing: {
        '3px': '3px',
      }
    }
  }
}
```

也可以直接在 `theme.inset` 或 `theme.extend.inset` 属性中定制

```js
// 📄 tailwind.config.js
module.exports = {
  theme: {
    extend: {
      inset: {
        '3px': '3px',
      }
    }
  }
}
```

:bulb: Top/Right/Bottom/Left/Inset 基础类支持任意值 `top-[3px]`

### Visibility
[Visibility 基础类](https://tailwindcss.com/docs/visibility)用以控制元素是否在页面可见

:bulb: 如果使用 `.invisible` 基础类，则元素会隐藏，但是它依然在页面保留占位空间。如果想完全隐藏元素，将从页面移除该 DOM 元素，可以使用 [`.hidden` 基础类](https://tailwindcss.com/docs/display#hidden)

### Z-index
[Z-index 基础类](https://tailwindcss.com/docs/z-index)用以控制元素的叠放顺序

:bulb: 支持在其前面添加 `-` 来设置为负值 `-z-50`

:bulb: Tailwind 提供了 6 个不同值的 Z-index 基础类，和一个 `.z-auto` 基础类，可以在 Tailwind 配置文件的 `theme.zIndex` 或 `theme.extend.zIndex` 属性中定制

```js
// 📄 tailwind.config.js
module.exports = {
  theme: {
    extend: {
      zIndex: {
        '100': '100',
      }
    }
  }
}
```

:bulb: 支持使用任意值 `z-[100]`

### Flexbox
Tailwind 针对 Flex 布局提供了多种基础类，以设置该布局中的不同元素的样式：

* [Flex Basis 基础类](https://tailwindcss.com/docs/flex-basis)用于设置 Flex 项目（子元素）的初始尺寸
  :bulb: 该基础类继承了 Spacing 基础类以及一系列的百分比的值，可以间接在 Tailwind 配置文件的 `theme.spacing` 或 `theme.extend.spacing` 属性中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        spacing: {
          '112': '28rem',
          '128': '32rem',
        }
      }
    }
  }
  ```

  也可以在 Tailwind 配置文档的 `theme.flexBasis` 或 `theme.extend.flexBasis` 属性中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        flexBasis: {
          '1/7': '14.2857143%',
          '2/7': '28.5714286%',
          '3/7': '42.8571429%',
          '4/7': '57.1428571%',
          '5/7': '71.4285714%',
          '6/7': '85.7142857%',
        }
      }
    }
  }
  ```

  :bulb: 支持任意值 `flex-basis-[14.2857143%]`

* [Flex Direction 基础类](https://tailwindcss.com/docs/flex-direction)用于设置 Flex 布局的方向
* [Flex Wrap 基础类](https://tailwindcss.com/docs/flex-wrap)用于设置 Flex 项目过多时其换行行为
* [Flex 基础类](https://tailwindcss.com/docs/flex)是设置简写属性 `flex` 的样式值

  :bulb: 可以在 Tailwind 配置文档的 `theme.flex` 或 `theme.extend.flex` 属性中进行定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        flex: {
          '2': '2 2 0%'
        }
      }
    }
  }
  ```

  :bulb: 支持任意值 `flex-[2_2_0%]`

* [Flex Grow 基础类](https://tailwindcss.com/docs/flex-grow)用于设置 Flex 项目伸长行为

  :bulb: 可以在 Tailwind 配置文档的 `theme.flexGrow` 或 `theme.extend.flexGrow` 属性中进行定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        flexGrow: {
          '2': 2
        }
      }
    }
  }
  ```

  :bulb: 支持任意值 `grow-[2]`

* [Flex Shrink 基础类](https://tailwindcss.com/docs/flex-shrink)用于设置 Flex 项目的压缩行为

  :bulb: 可以在 Tailwind 配置文档的 `theme.flexShrink` 或 `theme.extend.flexShrink` 属性中进行定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        flexShrink: {
          '2': 2
        }
      }
    }
  }
  ```

  :bulb: 支持任意值 `shrink-[2]`

* [Order 基础类](https://tailwindcss.com/docs/order) 用于设置 Flex 项目的次序

  :bulb: 可以在 Tailwind 配置文档的 `theme.order` 或 `theme.extend.order` 属性中进行定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        order: {
          '13': '13'
        }
      }
    }
  }
  ```

  :bulb: 支持任意值 `order-[13]`

* [Justify Content 基础类](https://tailwindcss.com/docs/justify-content)用于设置 Flex 或 Grid 布局中项目在主轴的对齐方式

* [Justify Items 基础类](https://tailwindcss.com/docs/justify-items)用于设置 Grid 布局的项目在水平方向的对齐方式

* [Justify Self 基础类](https://tailwindcss.com/docs/justify-self)用于设置 Flex 或 Grid 布局的项目内部的子元素在主轴方向的对齐方式

* [Align Content 基础类](https://tailwindcss.com/docs/align-content)用于设置 Grid 布局中项目在垂直方向的对齐方式

* [Align Items 基础类](https://tailwindcss.com/docs/align-items)用于设置 Flex 布局的项目在次轴的对齐方式

* [Align Self 基础类](https://tailwindcss.com/docs/align-self)用于设置 Flex 或 Grid 布局的项目内部的子元素在次轴方向的对齐方式

* [Place Content 基础类](https://tailwindcss.com/docs/place-content)是一个简写属性，用于同时设置 Flex 或 Grid 布局中项目在主轴和次轴的对齐方式，项目作为一个整体来进行设置

* [Place Items 基础类](https://tailwindcss.com/docs/place-items)是一个简写属性，用于同时设置 Flex 或 Grid 布局中项目在主轴和次轴的对齐方式，项目在其网格中分别进行设置

* [Place Self 基础类](https://tailwindcss.com/docs/place-self)是一个简写属性，用于同时设置 Flex 或 Grid 布局的项目内部的子元素在主轴和次轴方向的对齐方式

### Grid
Tailwind 针对 Grid 布局提供了多种基础类，以设置该布局中的不同元素的样式：

* [Grid Template Columns 基础类](https://tailwindcss.com/docs/grid-template-columns) 用于设置 Grid 容器中的列数量

  :bulb: Tailwind 默认提供了在 Grid 中最多创建 12 列的基础类，可以在 Tailwind 的配置文档的 `theme.gridTemplateColumns` 或 `theme.extend.gridTemplateColumns` 中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        gridTemplateColumns: {
          // Simple 16 column grid
          '16': 'repeat(16, minmax(0, 1fr))',

          // Complex site-specific column configuration
          'footer': '200px minmax(900px, 1fr) 100px',
        },
        gridColumnStart: {
          '13': '13',
          '14': '14',
          '15': '15',
          '16': '16',
          '17': '17',
        },
        gridColumnEnd: {
          '13': '13',
          '14': '14',
          '15': '15',
          '16': '16',
          '17': '17',
        }
      }
    }
  }
  ```

  :bulb: 支持任意值 `col-span-[span_16_/_span 16]`

* [Grid Column Start / End 基础类](https://tailwindcss.com/docs/grid-column)用于设置 Grid 项目横跨的列数，或者指定从哪一列开始，从哪一列结束

  :bulb: 可以在 Tailwind 的配置文档的 `theme` 或 `theme.extend` 下的 `gridColumn`、`gridColumnStart`、`gridColumnEnd` 属性中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        gridColumn: {
          'span-16': 'span 16 / span 16',
        }
      }
    }
  }
  ```

  :bulb: 支持任意值 `grid-cols-[200px_minmax(900px,_1fr)_100px]`

* [Grid Template Rows 基础类](https://tailwindcss.com/docs/grid-template-rows)用于设置 Grid 容器中的行数量

  :bulb: Tailwind 默认提供了在 Grid 中最多创建 6 行的基础类，可以在 Tailwind 的配置文档的 `theme.gridTemplateRows` 或 `theme.extend.gridTemplateRows` 中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        gridTemplateRows: {
          // Simple 8 row grid
          '8': 'repeat(8, minmax(0, 1fr))',

          // Complex site-specific row configuration
          'layout': '200px minmax(900px, 1fr) 100px',
        }
      }
    }
  }
  ```

  :bulb: 支持任意值 `grid-rows-[200px_minmax(900px,_1fr)_100px]`

* [Grid Row Start / End 基础类](https://tailwindcss.com/docs/grid-row)用于设置 Grid 项目横跨的行数，或者指定从哪一行开始，从哪一行结束

  :bulb: 可以在 Tailwind 的配置文档的 `theme` 或 `theme.extend` 下的 `gridRow`、`gridRowStart`、`gridRowEnd` 属性中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        gridRow: {
          'span-16': 'span 16 / span 16',
        },
        gridRowStart: {
          '8': '8',
          '9': '9',
          '10': '10',
          '11': '11',
          '12': '12',
          '13': '13',
        },
        gridRowEnd: {
          '8': '8',
          '9': '9',
          '10': '10',
          '11': '11',
          '12': '12',
          '13': '13',
        }
      }
    }
  }
  ```

  :bulb: 支持任意值 `row-[span_16_/_span_16]`

* [Grid Auto Flow 基础类](https://tailwindcss.com/docs/grid-auto-flow)用于设置 Grid 中项目的布局方向

* [Grid Auto Columns 基础类](https://tailwindcss.com/docs/grid-auto-columns)用于设置 Grid 布局中超出容器而隐式创建的网格的列宽

  :bulb: 可以在 Tailwind 的配置文档的 `theme.gridAutoColumns` 或 `theme.extend.gridAutoColumns` 属性中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        gridAutoColumns: {
          '2fr': 'minmax(0, 2fr)',
        }
      }
    }
  }
  ```

  :bulb: 支持任意值 `auto-cols-[minmax(0,_2fr)]`

* [Grid Auto Rows 基础类](https://tailwindcss.com/docs/grid-auto-rows)用于设置 Grid 布局中超出容器而隐式创建的网格的行宽

  :bulb: 可以在 Tailwind 的配置文档的 `theme.gridAutoRows` 或 `theme.extend.gridAutoRows` 属性中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        gridAutoRows: {
          '2fr': 'minmax(0, 2fr)',
        }
      }
    }
  }
  ```

  :bulb: 支持任意值 `auto-rows-[minmax(0,_2fr)]`

* [Gap 基础类](https://tailwindcss.com/docs/gap)用于设置 Grid 布局中网格之间的间隙

  :bulb: 该基础类继承了 Spacing 基础类以及一系列的百分比的值，可以间接在 Tailwind 配置文件的 `theme.spacing` 或 `theme.extend.spacing` 属性中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        spacing: {
          '11': '2.75rem',
        }
      }
    }
  }
  ```

  也可以在 Tailwind 配置文档的 `theme.gap` 或 `theme.extend.gap` 属性中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        gap: {
          '11': '2.75rem',
        }
      }
    }
  }
  ```

  :bulb: 支持任意值 `gap-[2.75rem]`

## 尺寸

### 间隔
* [Padding 基础类](https://tailwindcss.com/docs/padding)用于设置元素的内边距

  :bulb: 由于该基础类继承了 Spacing 基础类，所以可以间接在 Tailwind 配置文档的 `theme.spacing` 或 `theme.extend.spacing` 属性中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        spacing: {
          '5px': '5px',
        }
      }
    }
  }
  ```

  也可以直接在 Tailwind 配置文档的 `theme.padding` 或 `theme.extend.padding` 属性中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        padding: {
          '5px': '5px',
        }
      }
    }
  }
  ```

  :bulb: 支持任意值 `p-[5px]`

* [Margin 基础类](https://tailwindcss.com/docs/margin)用于设置元素的外边距

  :bulb: 支持在基础类前添加 `-` 将外边距设置为负值

  :bulb: 由于该基础类继承了 Spacing 基础类，所以可以间接在 Tailwind 配置文档的 `theme.spacing` 或 `theme.extend.spacing` 属性中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        spacing: {
          '5px': '5px',
        }
      }
    }
  }
  ```

  也可以直接在 Tailwind 配置文档的 `theme.padding` 或 `theme.extend.padding` 属性中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        margin: {
          '5px': '5px',
        }
      }
    }
  }
  ```

  :bulb: 支持任意值 `m-[5px]`

* [Space Between 基础类](https://tailwindcss.com/docs/space)用于设置元素之间的间隔（一般用于列表元素或 Flex 布局容器）

  :bulb: 可以在基础类前添加 `-` 将间隔设置为负值

  :warning: 该基础类实际上是为子元素（除第一个）添加外边距，并不能很好地处理复杂的布局，例如当元素过多，出现了换行，就可以造成对齐问题，此时应该使用 Grid 布局，然后用 [Gap 基础类](https://tailwindcss.com/docs/gap)为元素添加间隔。

  :bulb: 它不应该与 [Divide Width 基础类](https://tailwindcss.com/docs/divide-width)一起使用，因为两者在视觉上功能目的是一样的，都是为了区分并列的多个元素，这两个基础类应该是二选一来使用。

  :bulb: 由于该基础类继承了 Spacing 基础类，所以可以间接在 Tailwind 配置文档的 `theme.spacing` 或 `theme.extend.spacing` 属性中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        spacing: {
          '5px': '5px',
        }
      }
    }
  }
  ```

  也可以直接在 Tailwind 配置文档的 `theme.space` 或 `theme.extend.space` 属性中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        space: {
          '5px': '5px',
        }
      }
    }
  }
  ```

  :bulb: 支持任意值 `space-y-[5px]`

## 字体
### Font Family
[Font Family 基础类](https://tailwindcss.com/docs/font-family)用于设置元素文本内容所使用的字体家族

:bulb: Tailwind 默认提供了三组跨浏览器可适用的字体家族，分别无衬线字体 sans-serif、衬线字体 serif、mono 字体 monospaced。可以在 Tailwind 配置文档的 `theme.fontFamily` 属性中定制，字体家族可以是数组，也可以是字符串（各个字体用逗号分隔）

```js
// 📄 tailwind.config.js
module.exports = {
  theme: {
    fontFamily: {
      'display': ['Oswald', ...], // 数组形式
      'body': 'sans': 'Helvetica, Arial, sans-serif', // 字符串形式
    }
  }
}
```

:warning: 如果字体家族中，字体命名有非法值，应该使用双引号包括，或是使用转义符

```js
{
  // Won't work:
  'sans': ['Exo 2', ...],

  // Add quotes:
  'sans': ['"Exo 2"', ...],

  // ...or escape the space:
  'sans': ['Exo\\ 2', ...],
}
```

:bulb: 支持任意值 `font-['Open_Sans']`

### Font Size
[Font Size 基础类](https://tailwindcss.com/docs/font-size)设置元素文本内容的字体大小

:bulb: Tailwind 提供了 10 种字体尺寸的基础类，并为每种字体大小尺寸配上了合适的默认行号

:bulb: 可以在 Tailwind 配置文档的 `theme.fontSize` 属性中定制

```js
// 📄 tailwind.config.js
module.exports = {
  theme: {
    fontSize: {
      'tiny': '.875rem',
      'base': '1rem',
    }
  }
}
```

可以为不同的字体大小同时提供「配套」的行高，以二元数组的形式，第二个元素就是默认行高。第二个元素也可以是对象，其中属性 `lineHeight` 指定行高，还可以设置属性 `letterSpacing` 以提供默认的字间距

```js
// 📄 tailwind.config.js
module.exports = {
  theme: {
    fontSize: {
      base: ['16px', '24px'],
      sm: ['14px', {
        lineHeight: '20px',
      }],
      '2xl': ['24px', {
        letterSpacing: '-0.01em',
      }],
      '3xl': ['32px', {
        letterSpacing: '-0.02em',
        lineHeight: '40px',
      }],
    }
  }
}
```

:bulb: 支持任意值 `text-[14px]`

### Font Weight
[Font Weight 基础类](https://tailwindcss.com/docs/font-weight)设置元素的文本内容的字体的字重

:bulb: 可以在 Tailwind 配置文档的 `theme.fontWeight` 属性中定制

```js
// 📄 tailwind.config.js
module.exports = {
  theme: {
    fontWeight: {
      'extra-light': 100,
      semibold: 600,
      'extra-bold': 800,
      black: 900,
    }
  }
}
```

:bulb: 支持任意值 `font-[1100]`

### Font Style

### Font Variant Numeric
[Font Variant Numeric 基础类](https://tailwindcss.com/docs/font-variant-numeric)用于设置数字、分数和序号标记的字形

:bulb: 这个基础类是可组合的，即可以同时应用多个基础类

```html
<p class="ordinal slashed-zero tabular-nums ...">
  1234567890
</p>
```

:warning: 应用了基础类后，是否在页面起效果，还得看文本所使用的字体是否支持

:bulb: 使用 `.normal-nums` 基础类来重置所有字形设置

### Letter Spacing
[Letter Spacing 基础类](https://tailwindcss.com/docs/letter-spacing)用于设置字间距

:bulb: 可以在基础类前添加 `-` 将字间距设置为负值，但是这会降低文本的可读性

:bulb: 可以在 Tailwind 配置文档的 `theme.letterSpacing` 属性中定制

```js
// 📄 tailwind.config.js
module.exports = {
  theme: {
    letterSpacing: {
      tightest: '-.075em',
      normal: '0',
      wide: '.025em',
      widest: '.25em',
    }
  }
}
```

:bulb: 支持任意值 `tracking-[.25em]`

### Line Height
[Line Height 基础类](https://tailwindcss.com/docs/line-height)用于设置行高，提供了绝对行高和相对行高

:bulb: Tailwind 对于所提供的每一个默认字体大小都设置了配套的行高，可以再使用 Line Height 基础类来覆盖它

:bulb: 可以在 Tailwind 配置文档的 `theme.lineHeight` 属性中定制

```js
// 📄 tailwind.config.js
module.exports = {
  theme: {
    extend: {
      lineHeight: {
        'extra-loose': '2.5',
        '12': '3rem',
      }
    }
  }
}
```

:bulb: 支持任意值 `leading-[3rem]`

### Text Color
[Text Color 基础类](https://tailwindcss.com/docs/text-color)用以设置字体颜色

:bulb: Tailwind 默认提供的字体颜色基础类，都支持在其后添加表示透明度百分比的数值（与 [Opacity 基础类](https://tailwindcss.com/docs/opacity)所匹配） ，并用 `/` 分隔，也可以使用任意值 `text-blue-600/[.06]`

```html
<!-- 设置字体颜色透明度为 75% -->
<p class="text-blue-600/75">The quick brown fox...</p>
<!-- 设置字体颜色透明度为 6% -->
<p class="text-blue-600/[.06]">The quick brown fox...</p>
```

:bulb: 由于该基础类继承了 Color 基础类，所以可以间接在 Tailwind 配置文档的 `theme.colors` 或 `theme.extend.colors` 属性中定制

```js
// 📄 tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        'regal-blue': '#243c5a',
      },
    }
  }
}
```

也可以直接在 Tailwind 配置文档的 `theme.textColor` 或 `theme.extend.textColor` 属性中定制

:bulb: 支持任意值 `text-[#50d71e]`

## 背景
### Background Image
[Background Image 基础类](https://tailwindcss.com/docs/background-image)用以设置背景图

:bulb: Tailwind 默认所提供的基础类都是假设背景为线性渐变色，用于设定背景渐变色的方向 `.bg-gradient-{direction}`。可以在 Tailwind 配置文档的 `theme.backgroundImage` 或 `theme.extend.backgroundImage` 属性中定制，以设定背景为特定的图片

```js
// 📄 tailwind.config.js
module.exports = {
  theme: {
    extend: {
      backgroundImage: {
        'hero-pattern': "url('/img/hero-pattern.svg')",
        'footer-texture': "url('/img/footer-texture.png')",
      }
    }
  }
}
```

:bulb: 支持任意值 `bg-[url('/img/hero-pattern.svg')]`

如果背景设定的是渐变色，Tailwind 还提供了 [Gradient Color Stops 基础类](https://tailwindcss.com/docs/gradient-color-stops)用于指定不同渐变阶段 `from-{color}` 开始、`via-{color}` 中间、`to-{color}` 结束的具体颜色。如果 `to-{color}` 结束颜色没有指定，则默认为透明。

  :bulb: 由于该基础类继承了 Color 基础类，所以可以间接在 Tailwind 配置文档的 `theme.colors` 或 `theme.extend.colors` 属性中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        colors: {
          'regal-blue': '#243c5a',
        },
      }
    }
  }
  ```

  也可以直接在 Tailwind 配置文档的 `theme.gradientColorStops` 或 `theme.extend.gradientColorStops` 属性中定制

  :bulb: 支持任意值 `from-[#243c5a]`

### Background Size
[Background Size 基础类](https://tailwindcss.com/docs/background-size)用于设置背景图填充容器的方式，提供了 2 个基础类，分别是 `.bg-auto` 以图片原始大小来填充容器、`.bg-contain` 将图片进行保持宽高比的缩放，以将图片完整地包含在容器内（可能无法完全填满容器）、`.bg-cover` 将图片进行保持宽高的缩放，以用图片完整地填满容器

:bulb: 可以在 Tailwind 配置文档的 `theme.backgroundSize` 或 `theme.extend.backgroundSize` 属性中定制

```js
// 📄 tailwind.config.js
module.exports = {
  theme: {
    backgroundSize: {
      'auto': 'auto',
      'cover': 'cover',
      'contain': 'contain',
      '50%': '50%',
      '16': '4rem',
    }
  }
}
```

:bulb: 支持任意值 `bg-[length:200px_100px]` 由于 `bg-` 前缀所对应的基础类可以接受的值有多种，所以在任意值前需要添加提示 `length` 并用 `:` 冒号分隔，以提示 Tailwind 需要定制的是背景图片的尺寸大小

### Background Position
[Background Position 基础类](https://tailwindcss.com/docs/background-position)用于设置背景图先对于容器的位置

:bulb: 在 Tailwind 配置文档的 `theme.position` 或 `theme.extend.position` 属性中定制

```js
// 📄 tailwind.config.js
module.exports = {
```

* [Animation 基础类](https://tailwindcss.com/docs/animation)用于设置关键帧动画来实现过渡，Tailwind 提供了 4 种关键帧动画基础类：
  * `animate-spin` 旋转（可用于加载转圈圈的动效）
  * `animate-ping` 呼吸灯（可用于消息小红点的提醒）
  * `animate-pluse` 闪动（可用于网页/卡片的内容加载前，显示的骨架图案）
  * `animate-bounce` 跳动（可用于下载按钮动效，吸引用户点击）

  :bulb: 可以在 Tailwind 配置文档的 `theme.animation` 或 `theme.extend.animation` 属性中定制过渡，如果采用关键帧动画，则相应的关键帧的定义需要在 `theme.extend.keyFrames` 属性中设置

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        animation: {
          wiggle: 'wiggle 1s ease-in-out infinite',
        },
        keyframes: {
          wiggle: {
            '0%, 100%': { transform: 'rotate(-3deg)' },
            '50%': { transform: 'rotate(3deg)' },
          }
        }
      }
    }
  }
  ```

  :bulb: 支持任意值 `animate-[wiggle_1s_ease-in-out_infinite]`（相应的同名关键帧需要在全局样式里提供）

### Scroll
* [Scroll Behavior 基础类](https://tailwindcss.com/docs/scroll-behavior)用于设置页面的滚动效果

[Scroll Snap](https://developer.mozilla.org/zh-CN/docs/Web/CSS/CSS_Scroll_Snap) 是 CSS 的一个模块，对于容器内的元素进行滚动位置捕抓，并实现强制滚动定位，一般用于实现滚动吸附的效果。以下的基础类所设置的样式属性一般在触屏设备上效果更明显。

* [Scroll Snap Type 基础类](https://tailwindcss.com/docs/scroll-snap-type)用于设置容器内元素的滚动位置的捕抓方式：

  * 基础类 `.snap-x`、`.snap-y` 或 `.snap-both` 设置捕抓容器哪个方向上的滚动
  * 基础类 `.snap-mandatory`（总是强制吸附） 或 `.snap-proximity`（非强制，只有在达到特定的临界值才吸附）设置是否强制将最近的元素吸附到容器的指定位置

  以上两类 class 可以配合使用，以设定在哪个方向上实现那种方式的滚动吸附

* [Scroll Snap Align 基础类](https://tailwindcss.com/docs/scroll-snap-align)用于子元素上，以设置该元素吸附到容器的何处

* [Scroll Snap Stop 基础类](https://tailwindcss.com/docs/scroll-snap-stop)用于子元素上，当用户「大力」滑动时，是否允许多个子元素跳过吸附点（默认允许跳过），可以通过 `.snap-always` 来启用约束，这样吸附点对于元素都会实行吸附制动停止，避免一次「大力」滑动而滚动过多的元素。

* [Scroll Padding 基础类](https://tailwindcss.com/docs/scroll-padding)用于设置吸附点距离容器的偏移距离

  :bulb: 由于该基础类继承了 Spacing 基础类，所以可以间接在 Tailwind 配置文档的 `theme.spacing` 或 `theme.extend.spacing` 属性中定制

  ```js
  // 📄 tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        spacing: {
          '96': '24rem',
        }
      }
    }
  }
  ```

  也可以直接在 Tailwind 配置文档的 `theme.scrollPadding` 或 `theme.extend.scrollPadding` 属性中定制

  :bulb: 支持任意值 `scroll-p-[24rem]`

* [Scroll Margin 基础类](https://tailwindcss.com/docs/scroll-margin)用于设置元素距离吸附点的偏移距离

  :bulb: 由于该基础类继承了 Spacing 基础类，所以可以间接在 Tailwind 配置文档的 `theme.spacing` 或 `theme.extend.spacing` 属性中定制

  :bulb: 支持任意值 `scroll-m-[24rem]`

### Content
[Content 基础类](https://tailwindcss.com/docs/content)用于设置 `::before` 或 `::after` 伪元素的 `content` 属性的内容，Tailwind 提供了 `.content-none` 基础类，相应的样式值是 `content: "";` 这也是默认的样式值（空字符串）

:bulb: 支持任意值 `before:content-['Hello_World']`，这样就可以方便地将伪元素的 `content` 设置为任意字符串，例如通过 `attr()` 方法来读取元素的属性值，将其设置为 `content` 内容

```html
<div before="Hello World" class="before:content-[attr(before)]">
  <!-- ... -->
</div>
```

如果希望 `content` 中包含空格，应该在代码中使用 `_` 来替代表示

```html
<div class="before:content-['Hello_World']">
  <!-- ... -->
</div>
```

如果希望 `content` 中包含下划线，则应该在代码中使用转义符对其处理

```html
<div class="before:content-['Hello\_World']">
  <!-- ... -->
</div>
```

:bulb: 可以在 Tailwind 配置文档的 `theme.content` 或 `theme.extend.content` 属性中定制

```js
// 📄 tailwind.config.js
module.exports = {
  theme: {
    extend: {
      content: {
        'link': 'url("/icons/link.svg")',
      },
    }
  }
}
```
